# HD2 Preset Manager

헬다이버즈 2(Helldivers 2)의 스트라타젬(Stratagem) 및 부스터(Booster) 로드아웃 장비 프리셋을 효율적으로 설정하고 관리하기 위한 Electron + Vue.js 기반의 데스크톱 애플리케이션입니다.

## 🚀 프로젝트 실행 방법 (사용법)

본 프로젝트는 Node.js와 Python 환경이 준비된 상태에서 구동할 수 있습니다.

### 1. 의존성 패키지 설치
프로젝트 루트 폴더에서 아래 명령어를 실행하여 필요한 의존성 라이브러리를 설치합니다.
```bash
npm install
```

### 2. 개발 모드로 애플리케이션 실행
설치가 완료되면 아래 명령어를 통해 일렉트론 개발 서버와 함께 애플리케이션을 실행할 수 있습니다.
```bash
npm run electron:serve
```

### 3. 단축키 및 프리셋 적용법
- **프리셋 추가**: 우측 사이드 패널에서 `추가` 버튼을 눌러 새 프리셋을 생성할 수 있습니다.
- **스트라타젬/부스터 설정**: 아이콘을 우클릭하여 프리셋에 등록(최대 5개 선택)하고 좌클릭으로 필요 없는 아이템을 제외할 수 있습니다.
- **핫키 변경**: 하단 우측의 입력 단축키 버튼을 클릭한 뒤 원하는 키(예: `H`)를 누르면 해당 키가 매크로 실행 핫키로 등록됩니다.

## ⚠️ 중요 주의사항 및 안내 (필독)

본 프로젝트를 실행하거나 빌드하기 전에 아래의 안내 사항을 반드시 확인해 주시기 바랍니다.

> [!IMPORTANT]
> **1. 최신화 미흡으로 인한 실사용 불가 안내**
> - 본 프로젝트에 수집된 게임 내 자산 및 단축키 좌표 매크로 설정 데이터는 최신 버전으로 갱신되어 있지 않습니다.
> - 따라서 현재 버전의 게임에 즉각적으로 연동하여 실사용하는 것은 불가능합니다.

> [!TIP]
> **2. 개발자 코드 참조 권장**
> - 본 프로젝트를 활용하고 싶으신 개발자분들은 전체 프로그램을 패키징하여 직접 실행하기보다는, **핵심 기능 로직 코드만 부분적으로 참조하여 본인의 개인 프로젝트에 이식 및 커스텀 적용**시키는 것을 강력히 권장합니다.
> - **참조 권장 파일**:
>   - [src/background.js](src/background.js): `user32.dll` 라이브러리를 이용한 Windows 로우 레벨 키 입력 시뮬레이션 매크로 구동 핵심 로직
>   - [src/utils/logic.js](src/utils/logic.js): 프리셋 데이터의 보관, 예외 필터링 및 매크로 좌표 연산 알고리즘 로직
